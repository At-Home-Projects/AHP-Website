At Home Projects

At Home Projects is a small group run primaraly by High Schoolers and is for the creation and sharing of open sorce porjects such as school projects and other pieces of code that can easily be recreated at home! for example, the main subject of this group is, but not limited to, Arduino. We are open to other project types like Raspberry Pi projects, make code arcade, Roblox, websites, and anything else you can think of!

For bugs reports, help, sugestions, off topic conversations, and other things, join the Discord server.
